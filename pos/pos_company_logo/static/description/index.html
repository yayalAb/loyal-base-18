POS Company Logo

This module allows you to customize the logo displayed in the Odoo POS interface. It replaces the default Odoo logo with your company's branding to create a more professional and personalized experience.

Key Features
  • Replace default POS logo with your company logo.
  • Seamless integration with the Odoo POS frontend.
  • No technical skills required to apply the customization.
